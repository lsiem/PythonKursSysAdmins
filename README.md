## 🛠️ Lokale Installation

### 📋 Voraussetzungen

- 🐍 Python 3.x
- 📝 Ein Editor oder IDE mit Jupyter-Unterstützung
- 📦 Git

### 📚 Installationsschritte

1. **Repository klonen** 📥

   ```sh
   git clone https://github.com/lsiem/PythonKursSysAdmins.git
   cd PythonKursSysAdmins
   ```

2. **Virtuelle Umgebung erstellen und aktivieren** 🌿

   ```sh
   python3 -m venv venv
   source venv/bin/activate  # Für Unix oder macOS
   # oder
   .\venv\Scripts\activate  # Für Windows
   ```

3. **Benötigte Pakete installieren** 📦

   ```sh
   pip install -r requirements.txt
   ```

4. **Jupyter Notebook starten** 🚀

   ```sh
   jupyter notebook
   ```

5. **Fehlerbehebung** 🛠️

   - Stelle sicher, dass Python 3.x installiert ist.
   - Überprüfe, ob alle Pakete erfolgreich installiert wurden.
   - Stelle sicher, dass die virtuelle Umgebung aktiviert ist.

## ❓ FAQ

**1. Welche Python-Version benötige ich?**\
Benötigt wird Python 3.x. Es wird empfohlen, die neueste stabile Version zu verwenden.

**2. Kann ich die Notebooks offline nutzen?**\
Ja, nach der lokalen Installation kannst du die Notebooks ohne Internetverbindung verwenden.

**3. Was mache ich, wenn ein Notebook nicht startet?**\
Stelle sicher, dass alle benötigten Pakete installiert sind und deine virtuelle Umgebung aktiviert ist. Überprüfe außerdem die Python-Version.
